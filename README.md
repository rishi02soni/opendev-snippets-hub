## 🤝 How to Contribute

We welcome contributions from everyone! 🎉

### 🧾 Steps:

1. Fork this repository  
2. Clone your fork  
3. Add your snippet in:
```
/data/snippets.json
````

### 🧠 Format:

```json
{
  "title": "Your Snippet Name",
  "code": "Your Code Here"
}
```
4.Commit your changes
5.Push to your fork
6.Create a Pull Request 🚀
